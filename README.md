# AiSpect Website

This repository contains the source code for the official [AiSpect](https://github.com/QQDDTT/aispect-website) documentation website.

## Overview
AiSpect is a dual-track AI Agent Framework for Java 17+. This website is designed to provide comprehensive API references, user manuals, and download resources for developers worldwide.

## Internationalization (i18n)
The documentation is separated by language:
- `zh/`: 简体中文 (Simplified Chinese)
- `en/`: English
- `ja/`: 日本語 (Japanese)

## Local Development
This website is powered by Jekyll. To run it locally:
1. Ensure you have Ruby and Bundler installed.
2. Run `bundle install`.
3. Run `bundle exec jekyll serve`.
4. Visit `http://localhost:4000` in your browser.

## Deployment
The site is automatically built and deployed via GitHub Actions whenever changes are pushed to the `main` branch.
