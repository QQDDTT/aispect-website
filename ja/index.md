# AiSpect オープンプラットフォームへようこそ

AiSpect は、アノテーション駆動のクロスアーキテクチャ AI エージェント注入・オーケストレーションシステムです。シンプルなアノテーションを使用するだけで、従来の Java ビジネスコードに大規模言語モデル（LLM）の認知および推論能力を即座に付与することができます。
